Password Generator

Description

A desktop application for generating secure passwords with customizable parameters.
The project was developed as part of a university course in Java.

Features

- Custom password generation
  Users can choose password length, character types (uppercase, lowercase, digits, symbols), and even define their own character set.

- Automatic password generation
  Generate a strong password automatically by selecting only length and character types.

- Password strength check
  Basic evaluation of password strength.

Tech Stack

- Java
- Swing (GUI)

Installation & Run

1. Clone the repository:
   git clone https://github.com/daniyilamelin/PasswordGenerator

2. Open the project in your IDE (IntelliJ IDEA / Eclipse)

3. Run the main class:
   Main.java

Notes

This project demonstrates basic work with Java, GUI development using Swing, and simple application architecture.
